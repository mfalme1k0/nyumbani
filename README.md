# Nyumbani

Nyumbani is a dual-sided real estate platform for tenants and property owners/agents.

## Structure

- `backend`: Express API with PostgreSQL, JWT auth, RBAC.
- `frontend`: Next.js tenant/agent dashboard.

## Run locally

1. `npm install`
2. `cd backend && npm install`
3. `cd frontend && npm install`
4. Populate `.env`
5. `npm run dev:backend`
6. `npm run dev:frontend`
